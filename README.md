# 📖 [School_Library] <a name="about-project"></a>


**[School_Library] is a ruby library project, in which add students and teachers as users and then we track record of who borrows a specific book and when did he borrow.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

<details>
<summary>Database</summary>
  <ul>
    <li><a href="https://www.postgresql.org/">PostgreSQL</a></li>
  </ul>
  <summary>Ruby</summary>
</details>

### Key Features <a name="key-features"></a>

- Add Teachers and Students as Users
- Track records of borrowed books
